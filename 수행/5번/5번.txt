## 문제 5 (학번이름문제05)

**먼저 맨 위에 배열(또는 리스트) 입력 데이터 arr = [ ] 을 작성한 후**
**아래에 문제를 해결하는 파이썬 명령문을 작성한다.**

1. **배열(또는 리스트) arr[ ] 값을 입력 예시로 작성하고 arr[ ] 값을 줄에서**
2. **최댓값과 최솟값을 출력한다.**

- 입력 예시와 출력 예시를 보고 만드세요.

**입력 및 출력 예시:**
- 예시 1: arr = [23, 34, -23, 1, 4] → 최솟값: -23, 최댓값: 34
- 예시 2: arr = [100, 12, 35, 4, 77, 89, 7] → 최솟값: 4, 최댓값: 100
- 예시 3: arr = [25, 12, 89, -23, -77, 650, 56, 32, 100] → 최솟값: -77, 최댓값: 650
